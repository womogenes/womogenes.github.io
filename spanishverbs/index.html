This is a tool that conjugates Spanish verbs in various tenses.
It can conjugate verbs in the following tenses:

  • Present
  • Preterite
More tenses to come in March of 2020!

All of this runs in your browser; no APIs (if you know what those are) and no ads!
There's just lots and lots of JavaScript code to follow the rules of conjugation (with all the irregular forms as well).



This code is a JavaScript implementation of cliffpanos' Python Conjugator with some improvements.
